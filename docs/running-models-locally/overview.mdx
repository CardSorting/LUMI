<Warning>
**Not wired in this build.** `buildApiHandler` in `src/core/api/index.ts` only registers OpenRouter, ChatGPT Subscription, NousResearch, and Cloudflare Workers AI. Ollama and LM Studio handler files exist under `src/core/api/providers/` but are not connected to the provider dropdown (`src/shared/providers/providers.json`).

Use **OpenRouter** with a local OpenAI-compatible endpoint if you need self-hosted inference, or extend `createHandlerForProvider` to register Ollama/LM Studio.
</Warning>

## Handler files present (not active)

| Provider | Handler file |
|----------|--------------|
| Ollama | `src/core/api/providers/ollama.ts` |
| LM Studio | `src/core/api/providers/lmstudio.ts` |

Settings UI components exist under `webview-ui/src/components/settings/providers/` for when these providers are re-enabled.
