Welcome to the LUMI documentation. LUMI is a VS Code extension (`CardSorting.lumi`) that helps you build software through conversation, terminal access, and file edits — always with explicit approval.

## What is LUMI?

LUMI runs inside VS Code. It can read and write files, run shell commands, browse the web, connect MCP servers, and delegate work to subagents. Task history and cognitive memory use **BroccoliDB** locally; your approval gates every mutating action.

## Naming note

User-facing product: **LUMI**. Internal types retain the `DietCode` prefix (`DietCodeMessage`, `.dietcoderules/`). Docs use **LUMI** for product behavior and **DietCode** only for those identifiers.
