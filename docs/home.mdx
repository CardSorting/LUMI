Welcome to the LUMI documentation. LUMI is a VS Code extension that helps you build software through conversation, terminal access, and file edits — always with explicit approval.

## What is LUMI?

LUMI runs inside VS Code as `CardSorting.lumi`. It can read and write files, run shell commands, browse the web, connect MCP servers, and delegate work to subagents. Your codebase stays local; you approve every mutating action.

## Naming note

The user-facing product is **LUMI**. Internal types and storage keys still use the `DietCode` prefix from the upstream codebase (for example `DietCodeMessage`, `.dietcoderules/`). Documentation uses **LUMI** for product behavior and **DietCode** only when referring to those internal identifiers.
