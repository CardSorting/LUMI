Welcome to the DietCode documentation. Whether you're just getting started or looking to unlock advanced workflows, you'll find everything you need here.

## What is DietCode?

DietCode is an AI coding agent that lives in your editor and your terminal. It can read and write files, run terminal commands, use a browser, and help you build features through natural conversation. Every action requires your explicit approval. You're always in control.

> [!IMPORTANT]
> **Performance Optimized (v3.84.0)**: DietCode is now faster and more efficient, with significantly reduced startup times and improved history analysis.
