NousResearch provides the Hermes 4 family of models, focused on advanced reasoning and creative depth rather than optimizing purely for inference speed.

**Website:** [https://nousresearch.com/](https://nousresearch.com/)

### Getting an API Key

1.  **Sign Up/Sign In:** Go to the NousResearch inference platform. Create an account or sign in.
2.  **Navigate to API Keys:** Access the API key section.
3.  **Create a Key:** Generate a new API key.
4.  **Copy the Key:** Copy the API key immediately and store it securely.

### Supported Models

LUMI supports the following NousResearch models:

-   `Hermes-4-405B` (Default) - Largest Hermes 4 model focused on advanced reasoning and creative depth (128K context, $0.09/$0.37 per 1M tokens)
-   `Hermes-4-70B` - Balanced model for complex reasoning while staying fast and cost-effective (128K context, $0.05/$0.20 per 1M tokens)
-   `moonshotai/kimi-k2.6` - Frontier-scale multimodal model with 262K context, advanced vision, and agentic capabilities ($0.95/$4.00 per 1M tokens)
-   `xiaomi/mimo-v2-pro` - Flagship foundation model designed primarily for complex reasoning and production engineering.
-   `xiaomi/mimo-v2-omni` - Multimodal foundation model for agentic tasks requiring simultaneous text/image/video/audio processing.

### Configuration in LUMI

1.  **Open LUMI Settings:** Click the settings icon (⚙️) in the LUMI panel.
2.  **Select Provider:** Choose "NousResearch" from the "API Provider" dropdown.
3.  **Enter API Key:** Paste your NousResearch API key.
4.  **Select Model:** Choose your desired model from the "Model" dropdown.

### Tips and Notes

-   **Deep Reasoning Support**: Kimi K2.6 provides real-time thinking traces. LUMI automatically extracts and displays these as "Thinking" blocks.
-   **Vision Ready**: Models like `kimi-k2.6` and `mimo-v2-omni` support multimodal inputs for analyzing screenshots and UI designs.
-   **Prompt Caching**: ephemeral prompt caching is supported on next-gen models, significantly reducing latency and cost for long-context conversations.
-   **Agentic Capabilities**: The NousResearch provider is hardened for native tool calling, making it ideal for autonomous coding tasks.
