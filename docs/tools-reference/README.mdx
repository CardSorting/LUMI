## Maintainer notes

When adding a tool:

1. Add enum value in `src/shared/tools.ts`
2. Implement handler under `src/core/task/tools/handlers/`
3. Register in `ToolExecutorCoordinator.toolHandlersMap`
4. Update [all-dietcode-tools](/tools-reference/all-dietcode-tools) and [CODE_TO_DOC_MAP](/CODE_TO_DOC_MAP)

See [MAINTAINER](/MAINTAINER) for CI checks and branding rules.
